🎯 KEY FEATURES
================================================================================

1. THREE MILESTONES VISIBLE
   - 1. Complete All Lessons
     Progress: 0-50%
     Status: Shows X/Y lessons done
     Lock: Opens after watching videos + quizzes

   - 2. Pass Final Test (≥70%)
     Progress: 50-90%
     Status: Shows test percentage
     Lock: Opens after all lessons
     Retryable if failed

   - 3. Generate Certificate
     Progress: 90-100%
     Status: Ready or Generated
     Lock: Opens after passing test

2. REAL-TIME PROGRESS CALCULATION
   Overall = 0-50% (lessons) + 40% (test) + 10% (cert) = 0-100%

3. SMART UNLOCKING
   ✓ Final Test unlocks when: ALL lessons completed
   ✓ Certificate unlocks when: Final test score ≥ 70%

4. AUTO NEXT ACTION
   System tells student what to do next
   - "Complete X more lessons"
   - "Take the final test"
   - "Retry test (need 70%)"
   - "Generate your certificate"
   - "Course complete!"

5. COLOR-CODED STATUS
   - 🔒 Locked: Secondary badge
   - ⏱ In Progress: Warning badge
   - ✓ Completed: Success badge

6. PERSISTENT STATE
   - Data fetched from backend
   - Updates in real-time
   - Survives page refresh

================================================================================
🔄 TRACKING MECHANISM
================================================================================

Lessons Tracking:
  completedIds array
  → Tracks which lessons have quizzes completed
  → Updated when student submits quiz
  → Reset per user per course

Final Test Tracking:
  getFinalTestStatus(userId, courseId)
  → Returns: { completed, passed, score, percentage }
  → Stores in localStorage
  → Persists across sessions

Certificate Tracking:
  getCertificateByCourseid(userId, courseId)
  → Returns certificate object if generated
  → Stored in localStorage
  → Used for download

Visibility:
  Course Progress Component shows all three in real-time
  ├─ Fetches completion data
  ├─ Calculates overall progress
  ├─ Determines next action
  └─ Renders to UI

================================================================================
🔌 API USAGE
================================================================================

GET /api/courses/:courseId/progress/:userId
  Purpose: Get lessons completed
  Returns: { completed: number, total: number, percentage: number }

GET /api/users/:userId/courses/:courseId/final-test
  Purpose: Get final test status
  Returns: { completed, passed, score, percentage }

GET /api/users/:userId/certificates/:courseId
  Purpose: Get certificate (if generated)
  Returns: { id, score, percentage, completionDate, ... }
