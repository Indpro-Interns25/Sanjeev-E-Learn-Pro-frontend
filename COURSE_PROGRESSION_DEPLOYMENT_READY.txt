TIER 1: LESSONS (0-50% of course)

            ↓

┌──────────────────────────────────────┐
│ • Final Test button becomes enabled  │

│ • Score calculated                   │
│                                      │

│ → Certificate remains locked         │
│                                      │
│ IF score >= 70%:                     │
│ → Mark test as passed                │
│ → Progress jumps to 90%              │
│ → Certificate button enabled         │
│                                      │
│ Unlock Condition for Tier 3:         │
│ → Final test score >= 70%            │
└──────────────────────────────────────┘
            ↓

┌──────────────────────────────────────┐

└──────────────────────────────────────┘

================================================================================
📊 GATING LOGIC
================================================================================

Final Test Unlocks When:
  ✓ completedIds.length === lessons.length
  ✓ User has ProtectedRoute access
  ✓ User has student role
  Timeline: Automatic when last lesson completed

Certificate Unlocks When:
  ✓ finalTestResult.passed === true
  ✓ finalTestResult.percentage >= 70
  ✓ User role is student
  Timeline: Automatic when test submitted with passing score

Out-of-Order Prevention:
  ✓ Can't take final test → Redirected to lesson
  ✓ Can't generate cert → Button disabled
  ✓ Backend validates all conditions
  ✓ No ways to bypass progression
